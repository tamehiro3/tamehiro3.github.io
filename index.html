🌕 しのびのゲーム工房

家族で遊べる、無料のなぞときブラウザゲーム

「しのびのゲーム工房」は、子どもから大人まで安心して遊べる無料のなぞときゲームを公開している個人運営のサイトです。 どのゲームもアプリストアを通さず、スマホのブラウザで開いて「ホーム画面に追加」するだけで、アプリのように毎日遊べます。 課金要素はなく、名前やメールアドレスなどの個人情報を入力する場面もありません。一度読み込めばオフラインでも動作します。

公開中のゲーム

CNP にんじゃミッション工房

毎日とどく忍者ミッションで、里をそだてよう(小学生向け)

かわいい忍者キャラクターといっしょに、なぞなぞ・すいり・めいろ・あんごうなど10種類のなぞときに挑戦。 問題は日付をもとに毎日自動でつくられるので、あきずに続けられます。クリアのごほうびで自分だけの里を飾れます。

対象: 小学生〜 / 1回5分 / 制限時間なし / 保護者向け設定つき

CryptoNinja 忍びの謎巡り

毎日ひとつ、忍者と解く大人のなぞとき旅(40代からの脳トレ)

和風の世界を旅しながら、ずらし暗号・証言すいり・和文化クイズ・暦の謎など10種類の謎に挑む大人向けなぞときです。 文字は大きめ、制限時間もペナルティもなし。解いた旅は「旅日記」に残り、ほうびの小判で自分の里を育てられます。

対象: 大人(親子でも) / 1回3〜7分 / ヒント3段階 / 難易度3段階

▶ 遊んでみる

スマホでアプリのように遊ぶ方法

iPhoneSafariでゲームを開く → 共有ボタン → 「ホーム画面に追加」
AndroidChromeでゲームを開く → メニュー(⋮) → 「アプリをインストール」

ホーム画面のアイコンから起動すると全画面で遊べます。2回目からはオフラインでも動きます。

安心して遊べる工夫

  • 課金・ガチャ・スタミナはありません(すべて無料で最後まで遊べます)
  • 名前・メールアドレスなど個人情報の入力はありません(記録は端末の中だけに保存)
  • 自由に文字を打つチャット機能はありません
  • 暴力的・恐怖をあおる表現は使っていません
  • 子ども向けゲームからの外部リンクはありません
